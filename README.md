# QUIZ GAME

A quiz game using MERN stack.

Build with:
- React
- MongoDB
- NodeJS
- Express

## How to run

1. Clone the repo
2. Run the next:
```bash
cd frontend
npm i 
cd ..
cd backend
npm i
```
3. Create a `.env` file and add `MONGO_URI` variable to it.
4. Run:
```bash
npm run sample
npm run start
```
